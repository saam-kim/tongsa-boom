교사 활동 가이드

수업 진행 순서

  1. 교사 화면에서 학습 주제와 반을 선택합니다.
  2. QR 코드 생성하기 버튼을 누르면 전체화면 QR이 열려요.
  3. QR 화면을 교실 프로젝터/TV에 띄우세요.
  4. 학생들이 태블릿으로 QR 스캔 → 닉네임 입력 → 게임 시작!
  5. 모두 완료 후 Google Sheets에서 반별 기록을 확인합니다.
💡 QR 화면 안의 접속 주소를 직접 눌러도 교사가 학생 화면을 미리 체험할 수 있어요.

처음 설정 방법

1
게임 접속 URL 설정

이 게임 파일을 웹에 업로드하세요 (GitHub Pages 추천). GitHub Pages 주소는 보통 https://[아이디].github.io/[저장소명]/index.html 형식입니다. 이 주소를 '연동 설정 > 게임 접속 URL'에 입력하고 저장하면 QR 코드가 생성됩니다.

2
Google Sheets 연동 (선택)

설정하면 학생 플레이 결과가 스프레드시트에 자동 저장됩니다.

  1. Google Drive에서 새 스프레드시트를 만드세요.
  2. 상단 메뉴: 확장 프로그램 → Apps Script
  3. 기존 코드를 모두 지우고, config.js 파일 상단 주석 안의 ★ GAS 코드를 붙여넣으세요.
  4. 저장 (Ctrl+S) 후: 배포 → 새 배포 → 웹 앱
  5. 실행 계정: 나 / 액세스 권한: 모든 사용자 (익명 포함)로 배포
  6. 받은 URL을 '연동 설정 > Google Apps Script URL'에 입력하고 저장합니다.

학습 주제 추가 방법

data.js 파일을 텍스트 편집기로 열고, gameSets 배열에 새 주제를 추가합니다.

  • 각 주제는 id, title, description, categories로 구성됩니다.
  • 각 개념 유형에는 카드가 6개 이상 (짝수) 필요합니다.
  • data.js 파일 안의 주석 예시를 참고하세요.
💡 통합사회뿐 아니라 정치와 법, 경제, 사회문화 어느 과목에도 활용할 수 있어요.

게임 규칙 요약 (학생 안내용)

  1. 화면 상단의 개념 유형 버튼을 먼저 선택합니다.
  2. 그 유형에 해당하는 풍선 2개를 고릅니다.
  3. 정답이면 풍선이 터집니다. 오답이면 +3초 페널티!
  4. 모든 풍선을 없애면 기록이 랭킹에 등록됩니다.
